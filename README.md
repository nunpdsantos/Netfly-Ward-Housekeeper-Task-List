# Ward Housekeeper – Task List App

A simple single-page HTML application to support hospital ward housekeepers with clear **morning, midday, evening, ongoing and weekly tasks**.

The app is designed to:

- Make the daily workload easier to follow
- Help ensure food safety checks are completed
- Support accurate menu recording for patients
- Keep stock rotation and restocking organised

It is intended to be used on ward devices (e.g. iPads or desktop browsers).

---

## Features

- ✅ **Task cards** grouped by:
  - Morning
  - Midday
  - Evening
  - Ongoing responsibilities
  - Weekly responsibilities

- ✅ **Tick-off tasks**
  - Tap a card to mark it as done
  - Progress bar shows how much of the shift is complete
  - Progress is stored in the browser (localStorage)

- ✅ **Two view modes**
  - **Detailed view** – full descriptions of each task
  - **Compact view** – shorter view for quick checking

- ✅ **Mobile-friendly layout**
  - Works well on tablets and phones
  - Large tap targets for gloved hands
